## P0 — 核心体验缺失

### 1. 关键帧动画系统

**现状**：没有任何关键帧支持，元素属性（位置、缩放、旋转、透明度）在整个生命周期内恒定。

**实现路径**：
1. 类型定义 — `Keyframe` 接口（time, value, easing_type），追加到 `BaseTimelineElement`
2. TimelineManager 新增 `keyframes` 存取方法 + 插值计算（线性/贝塞尔）
3. VisualNode.render() 中根据当前时间插值变换属性
4. 属性面板增加关键帧按钮 + 关键帧编辑器 UI（曲线/折线图）
5. `PropertyGroup` 支持秒拍关键帧标记按钮

**工作量**：1.5–2 周

**影响范围**：`types/timeline.ts` → `core/timeline-manager.ts` → `renderer/visual-node.ts` → `properties/*.tsx`

---

### 2. 音频混音器

**现状**：只支持单轨道音量滑块和淡入淡出。

**功能范围**：
- 多轨道音频混合面板（增益 / 声像 / 静音 / 独奏）
- 音量包络线（在时间线上绘制音量曲线）
- 内置音效：压缩器、均衡器（EQ）、噪声门（Web Audio API）
- 音频波形放大显示 + 对齐辅助

**工作量**：2–3 周

---

### 3. 遮罩（Mask）与裁剪

**现状**：没有遮罩或裁剪功能，元素的可见区域不可控。

**最小实现**（1 周）：
- 矩形/椭圆裁剪到 `BaseTimelineElement`
- Canvas 渲染节点中用 `clip()` 实现
- 属性面板增加 Crop X/Y/Width/Height 控件

**进阶实现**（+1 周）：
- 贝塞尔路径遮罩
- 线性/径向渐变遮罩
- 反转遮罩

## P2 — 性能与架构

### 11. Canvas 渲染性能优化

**现状**：Canvas 2D API 渲染，每帧重建场景树。

**优化方向**：
- **脏区域检测**：只重绘有变化的区域
- **WebGL 加速**：基于 WebGL2 的渲染节点（滤镜等用 shader 实现）
- **节流/防抖**：空闲时低帧率，交互时高帧率
- **离屏缓存**：静态元素（图片、贴纸）渲染到离屏 canvas 缓存
- **内存管理**：视频帧缓存 LRU 淘汰策略

**工作量**：2–3 周

---

### 12. 存储层优化

**现状**：IndexedDB + OPFS，保存通过 `markDirty` + 节流写入。

**优化方向**：
- 增量保存（只保存变更的 scene/track）
- OPFS 作为主要文件存储，IndexedDB 做元数据索引
- 自动备份 & 崩溃恢复
- 项目缩略图优化（WebP 格式、渐进式）
- 存储配额监控与提示

**工作量**：1–1.5 周

---

### 13. 键盘快捷键系统增强

**现状**：有限的快捷键列表，硬编码在 `actions/definitions.ts`。

**需要做的**：
- 快捷键自定义 UI（Settings 面板中可重新绑定）
- 快捷键冲突检测
- 更多快捷键：数字键跳转标记点、字母键切换工具
- 游戏手柄支持（可选）

**工作量**：1 周

---

### 14. 命令系统改进

**现状**：Command 模式已实现，但部分操作不走 command（如 `updateSettings` 的 `pushHistory: false` 绕过）。

**改进**：
- 批量操作原子化（一个 Undo 撤销整批操作）
- Command 复合（CompoundCommand）
- 操作命名（显示在 UI 的 Undo 工具提示中）
- 操作历史面板（显示最近操作列表）

**工作量**：3–5 天

## 附录：技术债务

| 条目 | 位置 | 影响 | 建议 |
|------|------|------|------|
| Console.error 替代方案 | 所有 manager | 低 | 替换为统一 logger 或 toast |
| `as` 类型断言 | overlay-node.ts | 中 | 用类型守卫替代 |
| IndexedDB 版本管理 | storage/migrations | 低 | 当前机制健全，继续维护即可 |
| `useDeepCompareEffect` 开销 | preview 组件 | 中 | 场构建可增量更新 |
| OffscreenCanvas 兼容 | canvas-renderer.ts | 低 | 已有 fallback 到 HTMLCanvasElement |
| 缺少包入口导出类型 | packages/ui | 低 | 可导出共享图标类型 |
| Biome 排除 UI 目录 | 项目配置 | 低 | 考虑是否开启 lint |
