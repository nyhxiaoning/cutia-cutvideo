# Cutia 视频编辑器使用说明

Cutia 是一款**浏览器端运行、无需上传**的免费开源视频编辑器。所有媒体处理均在本地完成，保护您的隐私。

---

## 快速开始

### 1. 创建项目

1. 打开首页，点击 **创建项目**
2. 输入项目名称，选择画面比例（16:9 横屏 / 9:16 竖屏 / 1:1 方形等）
3. 进入编辑器主界面

### 2. 导入素材

在左侧**素材面板**中，切换到 **媒体** 选项卡：

- 点击 **导入文件** 选择本地视频、图片或音频文件
- 支持格式：MP4、WebM、MOV、PNG、JPG、GIF、MP3、WAV 等
- 素材仅存储在浏览器本地，**不会上传到任何服务器**

> 提示：你也可以使用 **音效** 选项卡搜索内置音效库。

### 3. 编辑视频

#### 添加素材到时间线

- 从素材面板将视频/图片/音频**拖拽**到下方时间线的轨道上
- 每个轨道可以放置多个素材，支持多轨道叠加

#### 时间线基础操作

| 操作     | 方法                                     |
| -------- | ---------------------------------------- |
| **裁剪** | 拖拽素材左右边缘调整时长                 |
| **分割** | 选中素材，点击工具栏剪刀图标，或按快捷键 |
| **移动** | 直接拖拽素材调整位置                     |
| **删除** | 选中素材，按 Delete 键或点击删除图标     |
| **复制** | 选中素材，点击复制图标                   |
| **书签** | 在时间标尺上点击添加标记点               |

时间线工具栏还支持：

- **吸附**：拖动素材时自动对齐到其他素材边缘（默认开启）
- **波纹编辑**：开启后删除素材，后面的内容自动前移填补空隙
- **缩放**：使用滑块或 +/- 按钮调整时间线显示比例

#### 添加文字

在素材面板切换到 **文字** 选项卡，选择文字样式拖入时间线：

- 双击预览画面编辑文字内容
- 在右侧属性面板调整字体、大小、颜色、对齐方式
- 支持文字描边、阴影、背景色等样式
- 预设文字样式一键应用

#### 添加贴纸

在素材面板切换到 **贴纸** 选项卡，选择贴纸拖入时间线。

#### 添加转场

在素材面板切换到 **转场** 选项卡，将转场效果拖拽到两段素材之间。

#### 添加滤镜/特效

在素材面板切换到 **滤镜** 或 **特效** 选项卡。

**滤镜**（颜色调整、模糊、艺术效果等）：

- 先在时间线上**选中**目标视频或图片素材
- 点击所需的滤镜效果，即可应用到所选素材上
- 再次点击同个滤镜可以移除
- 支持多种分类：颜色调整（黑白、复古、冷暖色调）、模糊效果、艺术风格、光照调节、动物/风光场景优化

**控制滤镜生效时长**：（Filter Range 滤镜配置）
应用滤镜后，可以在右侧属性面板中设置滤镜的起止时间，让滤镜只在素材的**指定时间段**内生效：

1. 选中已应用滤镜的素材
2. 在属性面板找到 **Filter Range** 分组
3. 设置 **Start（起始时间）** 和 **End（结束时间）**，单位为秒，相对于素材本身的时长
4. 预览时滤镜只在这个时间窗口内显示，素材的其他时段保持原画面

例如：一段 5 秒的视频，设置 Start=0、End=3，则前 3 秒显示黑白滤镜效果，后 2 秒恢复正常画面。不设置则滤镜覆盖整个素材时长。

**特效** 分为两种类型，使用方法不同：

**CSS 滤镜类特效**（需选中素材）：
包含：黑白（Grayscale）、复古（Sepia）、暖色（Warm）、冷色（Cold）、模糊（Blur）、马赛克（Heavy Blur/Mosaic）、戏剧（Drama）、发光（Glow）、黑白电影（Noir）、动物/风光场景优化等

1. 在时间线上**选中**一个视频或图片素材
2. 点击所需的特效，应用到该素材上
3. 再次点击同个特效可以移除
4. 时长控制：在右侧属性面板的 **Filter Range** 中设置起止时间

**粒子叠加层特效**（无需选中素材，直接点击）：
包含：下雨（Rain/暴雨/细雨）、下雪（Snow/暴风雪/小雪）、火花（Sparkle）、火焰（Fire）、烟雾（Smoke）、萤火虫（Fireflies）、气泡（Bubbles）、纸屑（Confetti）、屏幕抖动（Screen Shake）

1. 直接点击特效，系统会自动创建一个叠加层轨道
2. 特效覆盖在整个画面之上
3. 时长控制：直接在时间线上**拖拽**特效片段的左右边缘调整时长，或移动位置。**不支持** Filter Range

> **滤镜 vs 特效**：滤镜（Filters）选项卡主要提供人像美化、电影调色等高级色彩风格；特效（Effects）选项卡提供更丰富的 CSS 滤镜效果和粒子动画叠加层。

#### 全局色彩调整

在素材面板切换到 **调整** 选项卡，可以对整个画面施加全局色彩调整：

| 调整项 | 作用 | 调节范围 |
|--------|------|----------|
| **亮度 (Brightness)** | 整体画面明暗 | -100 ~ +100 |
| **对比度 (Contrast)** | 明暗反差程度 | -100 ~ +100 |
| **饱和度 (Saturation)** | 色彩鲜艳程度 | -100 ~ +100 |
| **色温 (Temperature)** | 暖色调/冷色调 | -100 ~ +100 |
| **暗角 (Vignette)** | 边缘暗角强度 | 0 ~ 100 |

- 调节滑块立即生效，可在预览中实时观察效果
- 点击「Reset All」一键重置所有调整为默认值
- 调整效果**影响整个画面**，而非单个素材，适用于后期调色

#### 添加字幕

在素材面板切换到 **字幕** 选项卡，可自动生成或手动添加字幕。

### 4. 调整素材属性

选中时间线上的素材，在右侧**属性面板**中进行精细调整：

| 素材类型      | 可调属性                                                               |
| ------------- | ---------------------------------------------------------------------- |
| **视频/图片** | 位置 X/Y、缩放、旋转、透明度、播放速度（视频）、滤镜时间范围、**属性关键帧动画**、**裁剪与遮罩** |
| **文字**      | 文字内容、字体、字号、颜色、描边、阴影、对齐、位置、缩放、旋转、透明度 |
| **音频**      | 音量、声像（Pan）、淡入淡出（Fade）、播放速度 |
| **贴纸**      | 位置、缩放、旋转、透明度                                               |

> **关键帧动画**：让素材的属性（位置、缩放、旋转、透明度）随时间变化，实现移动、缩放、旋转、淡入淡出等动态效果。使用方法：
> 1. 选中时间线上的素材
> 2. 将播放头拖到要设置关键帧的时间点
> 3. 在属性面板找到对应的属性（如 Scale、Position X/Y、Rotation、Opacity）
> 4. 点击属性旁边的菱形 ◇ 按钮（空心）→ 变为实心 ◆，表示该时间点已记录关键帧
> 5. **移动播放头**到另一个时间点，修改该属性的值，再点击 ◇ 添加第二个关键帧
> 6. 同理可添加更多关键帧，形成连续动画曲线
> 7. 播放预览：属性值在关键帧之间自动插值过渡
> 8. 点击已实心的 ◆ 会删除该时间点的关键帧
>
> > 支持缓动类型：关键帧之间默认线性过渡，可通过代码设置 ease-in/ease-out/ease-in-out（后续版本将在 UI 中开放选择）。

> **滤镜时间范围**：应用滤镜后，可以在属性面板中设置滤镜的起止时间，让滤镜只在素材的指定时间段内生效。
> 例如：前 3 秒黑白，后 2 秒暖色调。不设置则滤镜覆盖整个素材时长。

> **音频声像（Pan）**：控制音频在立体声场中的位置。在属性面板的 **Pan** 分组中拖动滑块：
> - 滑块向左（-1）→ 声音偏向左声道
> - 滑块向右（+1）→ 声音偏向右声道
> - 居中（0）→ 左右声道均衡，正常立体声
> - 滑块下方会实时显示 L（左偏）/ C（居中）/ R（右偏）指示
>
> 适合场景：多轨道音频时让不同音源处于不同声像位置，增强空间感（如左声道背景音乐、右声道人声旁白）。

> **音频淡入淡出（Fade）**：让音量在片段开始或结束时平滑过渡，避免突兀。在属性面板的 **Fade** 分组中设置：
> - **Fade In（淡入）**：设置时长（秒），播放时该片段前 N 秒音量从 0 逐渐上升到正常音量
> - **Fade Out（淡出）**：设置时长（秒），播放时该片段结束前 N 秒音量逐渐下降到 0
> - 设置为 0 则关闭淡入/淡出
>
> 适合场景：音乐片段开头设置 1-2 秒淡入避免突然响起；结尾设置 1-2 秒淡出让声音自然消失。

> 选中素材后，也可以在**预览画面**上直接拖拽调整位置和大小。

> **裁剪与遮罩（Crop & Mask）**：控制素材的可见区域和形状。在属性面板的 **Crop & Mask** 分组中设置：
> - **Crop X/Y**：裁剪起始位置（归一化 0~1，0=左上角，1=右下角），设置裁剪窗口的左上角坐标
> - **Crop Width/Height**：裁剪窗口大小（归一化 0~1，1=原始全宽/全高），缩小 = 只显示画面局部区域
> - **使用示例**：将 Crop X 设为 0.25、Crop Width 设为 0.5 → 只显示画面中间 50% 区域（左右各裁剪掉 25%）
> - **Mask（遮罩形状）**：选择 **Circle（圆形）** → 画面以圆形显示，四周透明；选择 **Rounded Rectangle（圆角矩形）** → 画面显示为圆角矩形
> - **Mask Radius（圆角半径）**：仅在选择圆角矩形时出现，控制圆角的大小（0=直角，100=最大圆角）
>
> 裁剪和遮罩可同时使用：先裁剪画面边缘再应用形状遮罩。适用于去除画面多余内容、制作头像圆形效果、圆角卡片等场景。
>
> **案例：制作圆形头像效果**
>
> 如果你有一段人物视频或图片，希望导出为圆形头像风格的画面，可以按以下步骤操作：
>
> 1. 将视频/图片素材拖入时间线，**选中**该素材
> 2. 在右侧属性面板找到 **Crop & Mask** 分组
> 3. 首先**裁剪画面**，去掉不需要的边缘：
>    - 观察预览画面，如果人物偏左，调整 **Crop X** 将裁剪窗口移动到合适位置（如 Crop X=0.2）
>    - 如果画面四周有多余背景，缩小 **Crop Width** 和 **Crop Height** 裁剪掉多余部分
>    - 例如：Crop X=0.1, Crop Y=0, Crop Width=0.8, Crop Height=1.0 → 左右各裁剪 10%
> 4. 然后**设置遮罩形状**：
>    - 将 **Mask** 下拉选择为 **Circle（圆形）**
>    - 此时预览画面中的视频/图片以圆形裁剪显示，四周变为透明
> 5. 如果希望是**圆角矩形**效果：
>    - 将 **Mask** 选择为 **Rounded Rectangle（圆角矩形）**
>    - 拖动 **Mask Radius** 滑块调整圆角大小，数值越大圆角越明显
> 6. 配合**位置和缩放**调整，让主体在圆形遮罩中居中显示
>
> 最终效果：视频/图片以圆形或圆角矩形呈现，适合用于头像、缩略图、圆角弹窗等场景。

### 5. 预览

中间**预览面板**实时显示当前时间线内容：

- **播放/暂停**：底部控制栏的播放按钮或按空格键
- **跳转**：点击时间线标尺任意位置，或拖动播放头
- **全屏**：点击全屏按钮
- **时间码**：点击可直接输入跳转到指定时间位置

### 6. 导出视频

点击顶部导航栏的 **导出** 按钮：

1. **选择格式**：
   - **MP4 (H.264)** — 兼容性更好，推荐
   - **WebM (VP9)** — 文件更小
2. **选择质量**：
   - 低 — 文件最小
   - 中 — 均衡
   - 高 — 推荐
   - 极高 — 文件最大
3. **音频**：可选择是否包含音频
4. 点击 **导出**，等待处理完成即可自动下载

> 导出全程在浏览器本地进行，无需等待上传。

### 7. 管理项目

点击左上角 Logo 按钮 → **退出项目** 返回项目列表页：

- **重命名**：点击顶部项目名称直接修改
- **删除**：在项目列表中删除不需要的项目
- 所有项目自动保存在浏览器 IndexedDB 中

## 常见问题

**Q: 项目数据存在哪里？会不会丢失？**
所有项目保存在浏览器本地 IndexedDB 中。建议定期导出最终视频。清理浏览器缓存前请确认项目已导出。

**Q: 支持导入哪些格式？**
视频：MP4、WebM、MOV；图片：PNG、JPG、GIF、WebP；音频：MP3、WAV、OGG、AAC、M4A。

**Q: 导出的视频有水印吗？**
完全没有。Cutia 导出的视频不含任何品牌标记。

**Q: 为什么导出比较慢？**
视频渲染在浏览器本地通过 FFmpeg.wasm 完成，速度取决于您电脑的 CPU 性能。较长的视频请耐心等待。
